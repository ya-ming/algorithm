# Collinear Points

[Write a program to recognize line patterns in a given set of points.
](https://coursera.cs.princeton.edu/algs4/assignments/collinear/specification.php)

In below example, the line segements will be:

* (1,7) -> (7,1)
* (1,7) -> (10,4)
* (3,5) -> (9,5)
